# Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** developed using **HTML, CSS, and JavaScript** to showcase personal information, technical skills, projects, achievements, resume, and contact details in a professional way.

This portfolio serves as an online presence for displaying work, experience, and project achievements while maintaining a clean and user-friendly design.

## 📌 Project Overview

The Personal Portfolio Website is designed to create a professional digital profile where users can present their skills, projects, education, certifications, and contact details.

The website provides an attractive interface with responsive design, making it accessible across desktops, tablets, and mobile devices.

## 🚀 Features

- Responsive website design
- Home page with introduction
- About Me section
- Skills showcase
- Projects section
- Resume/CV section
- Contact form
- Smooth scrolling navigation
- Interactive animations and effects
- Social media links integration

## 🛠️ Technologies Used

- **HTML5** – Website structure
- **CSS3** – Styling and responsiveness
- **JavaScript** – Interactive features and functionality
